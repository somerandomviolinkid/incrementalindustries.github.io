Incremental industries

By Bobby M, 2023

Welcome to incremental industries! First off you're going to want to mine some rock and chop some wood. Rock isn't very useful by itself, so you can crush it into minerals and other resources that are much more useful. Once you have some stone, you can smash some wood to get some sticks. Once you have sticks and some more wood, you can make some wood tools. Wood tools aren't very strong, but you can upgrade into better tools with stronger resources. Once you have a lot of stone, you can make your first primitive machine: a stone furnace. It isn't very powerful, but you can start smelting minerals into molten metals which will be very useful later on. Once you have some molten metals you can start making metal plates which will be useful for a number of uses, including for making machines, which can automate production of goods. In order to make machines you'll need a lot of machine hulls, circuits, and wires. Circuits require a lot of different components, including wires, transistors, and mechanical parts. In order to run machines however you'll need power, which can be made from a coal generator. In order to get a constant supply of resources, it's recommended to make a drilling machine which mines rock automatically. Making a circuit assembler, wiremill, and a metal casting machine will also be very useful, as circuits, wires, and plates will continue to be very important.
